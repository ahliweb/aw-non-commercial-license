# Security Policy

## Reporting Concerns

If you identify a security-sensitive issue, legal ambiguity with material impact, or repository misuse concern, please report it privately instead of opening a public issue.

## Contact

Please contact:

**commercial@ahliweb.com**

## Scope

This repository primarily contains license text and documentation. Security reports should focus on:

- repository integrity concerns
- misuse of official texts or examples
- impersonation of the canonical license source
- sensitive publication mistakes in repository materials
