# Contributing

Thank you for helping improve the clarity and quality of this repository.

## Scope of Contributions

Contributions are welcome for:

- typos
- formatting fixes
- readability improvements
- FAQ improvements
- example materials
- documentation clarifications

## Out of Scope Without Maintainer Approval

The following should not be changed without explicit maintainer review:

- legal meaning of the LICENSE text
- version name or numbering
- commercial contact workflow
- repository identity and canonical branding

## How to Contribute

1. Open an issue for clarification or proposed improvement.
2. Keep proposed wording precise and minimal.
3. Clearly distinguish documentation improvements from legal-term changes.
4. Submit pull requests with focused, reviewable changes.

## Legal-Term Changes

Any change that materially affects the legal meaning of the license should be treated as a versioned license change, not a minor wording fix.
