# Adoption Guide

## How to Use This License in Another Repository

1. Copy the official `LICENSE` text without modification unless you intentionally create a new forked license variant.
2. Add a short licensing note to your `README.md`.
3. Keep the license name exactly as published.
4. If you add project-specific contact details, make sure they are clearly maintained.
5. Avoid presenting the license as open source.
